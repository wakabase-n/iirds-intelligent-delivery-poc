# iiRDS Intelligent Delivery PoC

本リポジトリは、テクニカルコミュニケーションおよび情報設計（IA）における国際標準規格「iiRDS」を活用した、知的情報デリバリー（Intelligent Delivery）の実証実験（PoC）公式リソースページです。

コラムサイト「価値ある技術情報をiiRDSで届ける」等の記事と連動した検証用データ、および生成AI用プロンプトをプロジェクト単位で公開しています。

## 📁 ディレクトリ構成

### 🔹 01_columns/ (連載コラム連動)
連載記事と連動した検証用データ、および生成AI用プロンプトを月ごとに格納しています。
* **2026_07**: 属性の掛け合わせ（高度なパーソナライズ）実験用プロンプトとV3データセット
* **2026_08**: （次号予告）iiRDS Atomic Data v1.2 および ナレッジグラフ（V4）データ

### 🔹 02_ 〜 04_ (ユースケース・プロジェクト展開)
製造業の各ドメイン（半導体実装、XVL連携、部品メーカー等）における、iiRDS実装の実務的なPoCアセットを順次追加予定です。
